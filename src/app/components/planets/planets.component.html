I created a user-friendly platform inspired by Frontend Mentor's "The Planets" challenge. It showcases captivating images and detailed information about the planets in our Solar System.

In a departure from the original challenge, I improved the user experience by implementing a tab-based navigation system. I utilized Angular and Angular Material to streamline the creation of individual components for each planet, simplifying the routing process. This subtle yet significant change sets it apart from Frontend Mentor's version.

Objectives & Goals

My primary objective was to gain proficiency in migrating an Angular-built static website to Netlify and to re-create the challenge using this framework.

Problems and Thought Process

The project development went smoothly, but a significant issue arose just before the site's launch. Upon deploying the project to Netlify, I encountered a routing problem, realizing that managing routing with a framework introduced more complexity compared to not using one.

Fortunately, I found a comprehensive online guide with step-by-step instructions on correctly publishing an Angular project to Netlify. This resource helped me successfully resolve the routing issue and ensured a smooth launch for the site.

Lessons Learned

To ensure smooth functionality of our newly deployed Angular site with Netlify routing, the crucial steps involved customizing our angular.json file and creating a _redirects file within our src folder. This experience, being my second project uploaded to Netlify, has strengthened my confidence in the reliability of this hosting service.
