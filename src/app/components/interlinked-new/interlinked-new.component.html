Work

In contrast to its predecessor, the new Interlinked platform has undergone a significant upgrade by adopting the Laravel framework, replacing its previous custom MVC implementation. My primary focus was to enhance the UI of this new application.

I relocated the "Add Channel" button, originally situated under the tab links, to the right end of the layout. This adjustment aims to create a more aligned and visually cohesive appearance for improved user experience.

I eliminated the box shadow on this specific card to prevent redundancy since its parent card already features a shadow.

I repositioned the Order button to the right side of the container to achieve better alignment. Additionally, I removed the shadow from the green card, ensuring consistency with its parent card. To enhance user interaction, I introduced hover effects to the outline buttons positioned beneath it.

I enhanced the user experience by introducing a new empty state. In contrast to the previous version, which only featured plain text, the new empty state is designed to provide a more engaging and visually appealing experience.

I refined the layout for the "My Projects" title, as well as the "View Projects" and "30 Days Activity" buttons, ensuring improved alignment. Additionally, I adjusted the grid on the cards below to enhance overall responsiveness, optimizing the display for various screen sizes.

Objectives & Goals

The primary goal of my involvement in this project is to systematically evaluate the application's UI and implement changes that enhance the overall user experience. My role is centered on identifying areas where improvements can be made and applying those changes effectively.

Problems and Thought Process

Although the application leveraged Bootstrap 5, a notable issue arose with the sidebar, which was originally implemented using vanilla CSS.

This implementation posed challenges for mobile viewing, as it wasn't optimized for smaller screens. To address this, I focused on optimizing mobile responsiveness by reorganizing elements to stack vertically on majority of the pages.

Lessons Learned

Throughout this experience, I gained valuable insights into utilizing the SFTP extension for Visual Studio Code (VSCode). This knowledge has proven instrumental in streamlining file transfers and enhancing the efficiency of the development process.
